HOW TO TEST MESSAGING (SMS/WhatsApp)

Your endpoint /twilio-incoming handles BOTH SMS and WhatsApp.

SETUP IN TWILIO:
1. Go to Twilio Console → Phone Numbers → Manage → Active Numbers
2. Click your number
3. Under "Messaging" section, set:
   Webhook URL: https://leasing-copilot-mvp.onrender.com/twilio-incoming
   HTTP Method: POST
4. Save

TEST WITH SMS (if WhatsApp not available):
1. Send SMS to your Twilio number from your phone
2. Check backend logs for: "Message content: [your message]"
3. You should receive a reply via SMS

TEST WITH WHATSAPP (if enabled):
1. Send WhatsApp message to your Twilio number
2. Same process - check logs and verify reply

CHECK IF IT'S WORKING:
- Run: python test_data/test_whatsapp.py (checks config)
- Run: test_data/check_whatsapp.sql in Supabase (checks if messages saved)
